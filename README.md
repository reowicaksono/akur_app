
# Akur App

Application mobile with extraction feature and collab with gpt


## Wanna Contribute?

Preparation

```bash
  flutter clean
```
```bash
  delete .\pubspec.lock
```
```bash
  delete .\lib\generated_plugin_registrant.dart
``` 
```bash
  flutter pub get --verbose
``` 
##### note : please keep clean architecture & SOLID
## Running Tests



```bash
  flutter test
```
